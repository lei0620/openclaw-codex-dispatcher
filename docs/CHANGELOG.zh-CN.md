# 更新记录

## v1.8.0 - 安全存储版

- Android 上的 NAS 服务地址和访问密码改由 Android Keystore AES-GCM 加密保存，不再留在 WebView localStorage。
- 从旧版本升级时自动迁移现有地址和密码，迁移成功后删除旧明文，无需重新输入。
- 原生加密读取或写入失败时明确提示并保留旧数据，不会静默清空凭据。
- 浏览器访问 NAS 面板时继续使用浏览器本地设置，不影响网页版使用。

## v1.7.0 - 交互恢复版

- App 从后台回到前台、网络恢复或页面重新显示时，自动重连实时通道并校准状态，不会自动重发任务。
- 用户向上阅读历史时，新回复不再强制跳到底部；显示“回到最新”按钮供手动返回。
- 会话列表增加“执行中”“等授权”“排队”“失败”状态标记，切换并行任务更直观。
- 设置页增加连接诊断，可查看 NAS 延迟、实时连接、Win11 心跳、Codex 状态和当前会话映射。
- 一键复制诊断时自动隐藏 Token、密码、Authorization、WebDAV 凭据和密钥格式内容。

## v1.6.0 - 实时消息版

- 手机通过 NAS 专用 WebSocket 通道实时接收任务状态、回复、权限请求和电脑在线变化。
- 网络断开后按事件游标自动补发，游标过期时自动完整校准，不再依赖高频整页刷新。
- 手机发送后立即显示消息气泡；失败时可在原消息上重试，相同消息不会重复执行。
- 同一个对话或同一个绑定窗口串行排队，不同未绑定对话可以并行执行。
- 完整刷新由每 2 秒降低为每 30 秒兜底，减少滚动跳动、重复消息和 NAS 请求量。

## v1.5.0 - 稳定连接版

- 顶部连接图标可查看 NAS、Win11 电脑代理和 Codex 服务三层真实状态。
- 兼容新版 `ChatGPT.exe` 和旧版 `Codex.exe` 桌面宿主，并排除辅助小窗口和命令行进程。
- Win11 代理会预热并持续检查 Codex 会话服务，异常后自动恢复。
- 手机会话固定走对应的 Codex 会话通道，服务不可用时明确失败，不再回退到可能发错频道的方式。
- 关闭 Android 原生桥接请求日志，避免访问凭据出现在调试日志中。

## v1.3.0 - 自动激活版

- 飞牛 NAS 重启后自动执行 `docker compose up -d`，恢复手机面板服务。
- Win11 登录后自动常驻监听器，打开 Codex 客户端后代理自动上线。
- 手机设置页显示当前版本和本版更新内容。
- 保持 GitHub 源码去敏：不提交真实访问密码、WebDAV 密码、本地配置和 APK 历史包。
